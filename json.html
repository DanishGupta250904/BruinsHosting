Introduction to JSON

JSON (JavaScript Object Notation) is a lightweight data interchange format that is easy for humans to read and write, and easy for machines to parse and generate. It is based on a subset of the JavaScript Programming Language, and it uses a key-value pair to store data.

JSON Syntax

{
    "name": "John Doe",
    "age": 30,
    "email": "john.doe@example.com"
}

Reading JSON from a File

To read JSON data from a file, we can use the fetch API or the XMLHttpRequest object. Here's an example using the fetch API:

fetch('data.json')
    .then(response => response.json())
    .then(data => console.log(data));

Reading JSON from an Online Source

To read JSON data from an online source, we can use the fetch API. Here's an example:

fetch('https://jsonplaceholder.typicode.com/todos/1')
    .then(response => response.json())
    .then(data => console.log(data));
