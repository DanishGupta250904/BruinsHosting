Angular Features

Component

Components are the building blocks of Angular applications. They are reusable, self-contained pieces of UI with their own HTML templates and logic.

Example:

            
                import { Component } from '@angular/core';

                @Component({
                  selector: 'app-root',
                  template: '<p>This is an root component</p>'
                })
                export class RootComponent {}

Explanation:

  • import { Injectable } from '@angular/core';: This line imports the Injectable decorator from the Angular core library.
  • @Injectable({...}): This is a decorator that provides metadata about the service.
  • providedIn: 'root': This specifies that the service should be provided at the root level of the application.
  • export class RootService {}: This defines the service class.
  • constructor() { }: This is the constructor method of the service class. It is called when an instance of the service is created.


Routing

Routing in Angular allows you to navigate between different components based on the URL. It enables single-page application behavior by loading components dynamically without full page reloads.

Example:

            
                import { NgModule } from '@angular/core';
                import { Routes, RouterModule } from '@angular/router';
                import { HomeComponent } from './home.component';
                import { AboutComponent } from './about.component';

                const routes: Routes = [
                { path: '', component: HomeComponent },
                { path: 'about', component: AboutComponent }
                ];

                @NgModule({
                imports: [RouterModule.forRoot(routes)],
                exports: [RouterModule]
                })
                export class AppRoutingModule { }

Explanation:

  • import { NgModule } from '@angular/core';: This line imports the NgModule decorator from the Angular core library.
  • import { Routes, RouterModule } from '@angular/router';: This line imports the Routes interface and RouterModule from the Angular router module.
  • const routes: Routes = [ ... ];: This defines an array of route configurations. Each route maps a URL path to a component.
  • @NgModule({ ... }): This is a decorator that provides metadata about the NgModule. It imports the RouterModule and configures it with the defined routes using forRoot().
  • export class AppRoutingModule { }: This exports the AppRoutingModule class so that it can be imported and used in other modules.
