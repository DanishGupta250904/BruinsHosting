1. document.getElementById():-

In this our main motive will be to fetch the data from the authorized ID given to the fucntion , it will get the data only of that Id

This is the paragraph 1 which is having the Id "intro1".

This is the paragraph 2 which is having the Id "intro2".

JavaScript

            
                function CallByID()
                {
                    const elements = ["intro1", "intro2", "intro3"];
                    let result = " ";
                    for (let i = 0; i < elements.length; i++) {
                        const myElement = document.getElementById(elements[i]);
                        result += myElement.innerHTML + "
";
                     }
                    document.getElementById("id").innerHTML = result;
                }

So after clicking on fetch data you can see we get the data , which was having id intro1,intro2,intro3,with this you can even add mutiple ids at same time to fetch the data
